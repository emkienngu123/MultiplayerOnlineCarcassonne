
# carcassonne
Carcassonne with networking multiplayer , RL and MCTS  implementation in python
## Features

* Tilesets 
    * Base game
    * The river
    * Inns and cathedrals
* Abbots
* Farmers

## Installation

* Clone the project
* Go to the project folder
* Run: 

	    pip install .

You can now use the API in other projects.

## API
* Run the game:
	`python .\main.py`
